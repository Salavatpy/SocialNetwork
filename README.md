# SocialNetwork
Test project for the backend of little social network
Realized:
  Models:
    Profile
    Post
  Methods:
    like
    unlike
    login
    signup
    update profile data
To start the project open project, enter the required folder by comand line and enter python manage.py runserver.
After that in your browser enter http://127.0.0.1:8000/ and use the funcionality of project. Please do not be strict, I spend in total 6 hour this project.
